EJEMPLO DE SVG

Empezemos

 Si quieres empezar con SVG tienes que saber que esto se puede hacer desde varios programas
pero vamos a hacerlo desde el navegador como nos lo permite hacerlo html.
(+info)
  1. < svg > Esta etiqueta guarda la figura que estemos dispuestos a crear
  2. < text > Con esta etiqueta crearemos un texto en versión svg
  3. < rect > Con esto crearemos rectangulos

Hay muchas etiquetas de formas en este lenguaje, te invito a que investigues en w3school
