Historia

El organismo W3C (World Wide Web Consortium),
encargado de crear todos los estándares relacionados con la web,
propuso la creación de un lenguaje de hojas de estilos específico para el lenguaje HTML y se presentaron nueve propuestas.
Las dos propuestas que se tuvieron en cuenta fueron la CHSS  y la SSP.
