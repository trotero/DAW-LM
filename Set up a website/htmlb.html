Historia

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN
propuso un nuevo sistema de "hipertexto" para compartir documentos.
Tras finalizar el desarrollo de su sistema de "hipertexto",
Tim Berners-Lee lo presentó a una convocatoria organizada para crear un sistema de "hipertexto" para Internet.
Después de unir sus fuerzas con el ingeniero de sistemas Robert Cailliau,
presentaron la propuesta ganadora llamada WorldWideWeb (W3).

Empezemos

 Si quieres empezar con HTML tienes que tener en cuenta las principales etiquetas
tales como :
  1. < html > Esta es la que guarda toda la pag. web
  2. < head > Esta es la que guarda etiquetas que se tienen que cargar rapido, como el favicon,style etc..
  3. < body > Esta sería la que guardaría todo el contenido
